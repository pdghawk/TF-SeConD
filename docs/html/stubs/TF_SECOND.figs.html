TF_SECOND.figs.plot_Pkt(data_dict, file_path, direction, logscale=False, save_figs=True, plot_figs=False)¶

Color plot of P_k(t) along either x or y axis in k, over time

A 2d false color plot of P_{k_x}(t) (or P_{k_y{(t)}), with the resultant axes being k_{x}(or k_{y}) and t, and the color representing microscopic polarization at that point. P can be plotted on linear or log scale.

Parameters:
  • data_dict – a data dictionary produced by TF_SECOND.solve_SBE
  • file_path – directory to save to - figure will be saved to file_path/Pkt…..pdf
Keyword Arguments:
 
  • logscale – (bool) if True P will be plotted on log10(P) scale
  • save_figs – (bool) save figures to file if True
  • plot_figs – (bool) plot figures to screen if True
TF_SECOND.figs.plot_absorption(data_dict_list, file_path, labels=[], save_figs=True, plot_figs=False)¶

plot a scan of absorption data for different experiments

An simulation/experiment for absorption can be run with TF_SECOND.calculate_absorption with outputs a data dictionary. One can run a series of simulations for different conditions, (e.g. grid, laser parameter etc), and save the dictionaries into a python list. This funtion will plot the absorption from each experiment, with color coded lines and legend.

Parameters:
  • data_dict – a python list of dictionaries output by TF_SECOND.solve_SBE
  • file_path – directory to save to - figure will be saved to file_path/absorption.pdf
Keyword Arguments:
 
  • labels – python list of strings, labels for legend corresponding to each dictionary in the data_dict_list list
  • save_figs – (bool) save figures to file if True
  • plot_figs – (bool) plot figures to screen if True
TF_SECOND.figs.plot_spectra(data_dict_list, file_path, labels=[], save_figs=True, plot_figs=False)¶

plot a series of spectra, given a list of data dictionaries from solve_SBE

Harmonic sideband spectra can be calculated using TF_SECOND.solve_SBE, which outputs a dictionary of useful quantiites. Saving a series of such dictionaries to a python list, can then be passed to plot_spectra, which will plot the spectra for all cases in the list, with lines colored according the position of the data in the python list. This is particularly useful for scanning a single parameter of an experiment, and seeing how the results vary.

If a single data dictionary is passed, the spectra for that single experiment will be plotted in black.

Parameters:
  • data_dict – a python list of dictionaries output by TF_SECOND.solve_SBE
  • file_path – directory to save to - figure will be saved to file_path/spectra.pdf
Keyword Arguments:
 
  • labels – python list of strings, labels for legend corresponding to each dictionary in the data_dict_list list
  • save_figs – (bool) save figures to file if True
  • plot_figs – (bool) plot figures to screen if True

TF_SECOND.figs.plot_thz_polar_spectra(data_dict_list, thz_pols, file_path, save_figs=True, plot_figs=False)¶

polar plot of spectra as THz laser polarization rotated

Produces a false colour image of how the harmonic sideband spectra varies as THz laser polarization is roatated. Plot has axes: r: sideband order theta: THz polarization angle w.r.t to crystal ( 0rad: X-Gamma-X) color represents log10(I(w)) normalised to second order sideband.

Parameters:
  • data_dict_list – a data dictionary produced by TF_SECOND.solve_SBE
  • thz_pols – numpy array of the polarization angle of THz laser, data_dict_list[i] should have data corresponding to polarization in thz_pols[i]
  • file_path – directory to save to - figure will be saved to file_path/THz_polar_spectra.pdf
Keyword Arguments:
 
  • save_figs – (bool) save figures to file if True
  • plot_figs – (bool) plot figures to screen if True
