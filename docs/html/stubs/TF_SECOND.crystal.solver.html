TF_SECOND.crystal.solver¶

class TF_SECOND.crystal.solver(XTAL_PARAMS, LASER_PARAMS, GRID_PARAMS)¶

For the calculation of dynamics in two dimensional semiconductors

OVERVIEW:

This class provides methods for the calculation of absorption and dynamics in two dimensional semiconductors.

A guide for how to use this class, and other parts of the TF_SeConD package, see the User Guide in the docs.

Parameters of the model should be passed at initialization. These parameters are passed as dictionary objects. The script get_params.py provides a useful tool for simple creation of these dictionary objects.

The dynamical problem in solved using the semiconductor optical Bloch equations. Further details of the background physics documentation under Background.

Tensorflow is used in several of the methods to allow for utilzation of GPUs to increase perfmonance of large array calculations.

calculate_absorption(include_coulomb=True, save_graph=False)¶

calculate polarization response to get absorption

Solve the semiconductor dynamics for the system. Must first find the 1s state via the variational method - “variational_1s()”. This method includes only the optical field. For calculations with Thz driving field one can use “solve_SBE()”. The Coulomb interaction is included by default. It can be optionally turned off. In this case the dephasing reverts to a simple phenomenological form that doesn’t tak the 1s shape into account.

Keyword Arguments:
 
  • include_coulomb – True will include the Coulomb interaction, False won’t. Defaults to True.
  • save_graph – Save the graph to a tensorboard file. Defaults to False
Returns:

dictionary of:
  • freq_dict:
    Dictionary of variables that are functions of frequency :
    • Pw: macroscopic polarization in frequency,
    • Ew: optical field spectrum,
    • w_norm: frequency grid in exciton units, centered at gap energy,
    • w_eV: frequency grid in units of eV,
    • absorption: absorption normalized to max 1
    • absorption_bare: absorption without normalization
  • time_dict:
    Dictionary of variable that are functions of time:
    • opt_field: optical field in time,
    • opt_env: optical envelope in time,
    • polarization: macroscopic polarization in time
Return type:

(dictionary)

solve_SBE(include_coulomb=True, save_graph=False)¶

Solve the semiconductor optical Bloch equations

Solve the semiconductor dynamics for the system. Must first find the 1s state via the variational method - “variational_1s()”. This method includes both the optical and THz fields. For calculations with only an optical field one can use “calculate_absorption()”. The Coulomb interaction is included by default. It can be optionally turned off. In this case the dephasing reverts to a simple phenomenological form that doesn’t tak the 1s shape into account.

Keyword Arguments:
 
  • include_coulomb – True will include the Coulomb interaction, False won’t. Defaults to True.
  • save_graph – Save the graph to a tensorboard file. Defaults to False
Returns:

dictionary of:
  • freq_dict:
    Dictionary of variables that are functions of frequency :
    • Pw: macroscopic polarization in frequency,
    • Ew: optical field spectrum,
    • w_eV: frequency grid in units of eV,
    • w_sideband: frequency grid in units of sideband orders
  • time_dict:
    Dictionary of variable that are functions of time:
    • opt_field: optical field in time,
    • opt_env: optical envelope in time,
    • polarization: macroscopic polarization in time,
    • timegrid: the time grid
  • kt_dict:
    Dictionary of variables for plotting polarization in k and t:
    • kx: values of k along X-Gamma-X
    • ky: values of k along Y-Gamma-Y
    • Ax: laser vector potential in x direction at all times
    • Ay: laser vector potential in y direction at all times
    • P_kt_XGX: microscopic polarization along X-Gamma-X in time
    • P_kt_YGY: microscopic polarization along Y-Gamma-Y in time,
    • Pmicro2d: microscopic polarization in 2d k-space at a few times,
    • tlist2d: times at which Pmicro2d polarization saved
    • Ax_list2d: values of A_x at times in tlist2d
    • Ay_list2d: values of A_y at times in tlist2d
Return type:

(dictionary)
variational_gs(save_meta=False, save_summary=False)¶

Calculate 1s by energy minimization approx

The 1s exciton state is the lowest energy eigenstate of the system Hamiltonian. This means one can use a gradient descent approach to find the state of lowest energy - and this state will be the 1s exciton state. The quantity to be minimized is E=<phi|H|phi>/<phi|phi>, where phi is the state to be varied, and E is the energy (binding relative to gap). The gradient descent is performed by defining a tensorflow graph, with an ADAM optimizer routine.

Keyword Arguments:
 
  • save_meta (optional) – save metadata of the tensorflow graph to file for use in tensoboard. Defaults to False.
  • save_summary – save a summary of the loss decay to tensorboard output. Defaults to False.
Returns:

dictoinary of:
  • energy_1s_var: binding energy of 1s state
  • steps_taken: how many variational steps were taken
Return type:

(dictionary)
