TF_SECOND.runs.absorption_loop(Nx, Ny, filename, plot_figs=True, save_figs=False, do_symmetric=False, do_constrained=True)¶

absorption for different coulomb interaction models

This function will run a set of absorption simulations, for different coulomb interaction models and (optionally) save the figures to file, and (optionally) print them to screen. Compares, no Coulomb interaction (free carrier) with linear and ‘slab’ dielectric forms of the permitivity.

Parameters:
  • Nx – (int) x dimension of k grid to run
  • Ny – int) y dimension of k grid to run
  • filename – (string) directory name to save figures to - will save in this directory within ‘figs’ directory in current working directory
Keyword Arguments:
 
  • plot_figs – (bool) plot the figures? True or False
  • save_figs – (bool) save fhe figsures?
  • do_symmetric – (bool) symmetrise the crystal?
  • do_constrained – (bool) apply screening to Coloumb
Returns:

dummy result of 0 for testing only
TF_SECOND.runs.k_loop(Nx_arr, Ny_arr, filename, plot_figs=True, save_figs=False, do_symmetric=False, do_constrained=True, do_linear_eps=False, do_absorption=False)¶

vary the size of the k grid

This function will run a set of sideband generation or absortpion simulations, and (optionally) save the figures to file, and (optionally) print them to screen. This can be used to ensure convergance of the simulations.

Parameters:
  • Nx_arr – numpy array (1d) of x dimensions of k grid to run
  • Ny_arr – numpy array (1d) of y dimensions of k grid to run
  • filename – (string) directory name to save figures to - will save in this directory within ‘figs’ directory in current working directory
Keyword Arguments:
 
  • plot_figs – (bool) plot the figures? True or False
  • save_figs – (bool) save fhe figsures?
  • do_symmetric – (bool) symmetrise the crystal?
  • do_constrained – (bool) apply screening to Coloumb
  • do_linear_eps – (bool) enforces linear perimitivity of crystal in k if True
  • do_absorption – (bool) studies absorption problem if True
Returns:

dummy result of 0 for testing only
TF_SECOND.runs.thz_loop(thz_pols, Nx, Ny, filename, plot_figs=True, save_figs=False, do_symmetric=False, do_constrained=True, do_linear_eps=False, do_absorption=False)¶

vary the polarization of the THz field

This function will run a set of sideband generation simulations, for different THz laser field polarizations and (optionally) save the figures to file, and (optionally) print them to screen. This studies the effect of crystal asymmetry along different crystal axes.

Parameters:
  • thz_pols – numpy array of thz polarizations (in degrees), typically 0-90
  • Nx – (int) x dimension of k grid to run
  • Ny – int) y dimension of k grid to run
  • filename – (string) directory name to save figures to - will save in this directory within ‘figs’ directory in current working directory
Keyword Arguments:
 
  • plot_figs – (bool) plot the figures? True or False
  • save_figs – (bool) save fhe figsures?
  • do_symmetric – (bool) symmetrise the crystal?
  • do_constrained – (bool) apply screening to Coloumb
  • do_linear_eps – (bool) enforces linear perimitivity of crystal in k if True
  • do_absorption – (bool) studies absorption problem if True
Returns:

dummy result of 0 for testing only
