TF_SECOND.crystal.generate_params¶

TF_SECOND.crystal.generate_params(Nx=61, Ny=41, dt0_fs=0.08, symmetric=False, do_nonperiodic=False, do_constrained=True, do_linear_eps=False, do_absorption=False)¶

Creates a set of parameter dictionaries

this script sets up the parameters of the problem, can be called within other scripts to allow for easy variation of parameters and keeping the main run codes clean.

Keyword Arguments:
 
  • Nx – number of grid points in x direction
  • Ny – number of grid points in y direction
  • dt0_fs – spacing of points on the time grid
  • symmetric – make band structure symmetric in x&y
  • do_constrained – force a 2d nature to the Coulomb interaction (eps_r k dependent)
  • do_linear_eps – True make eps_r(k) linear in k (Keldysh approx). False has Trolle form [Trolle et al., Scientific Reports 7 (2017)]
  • absorption (do) – True if want to study absorption - will set parameters optimally, namely the central frequency and spectral width of the optical pulse, no ThZ pulse.
Returns:

tuple containing:

XTAL_PARAMS: parameters of the crystal (dictionary) LASER_PARAMS: paramaters of the laser (dictionary) GRID_PARAMS: parameters of the numerical grids (dictionary)
Return type:

(tuple)
